FishTank: AI-Powered Decentralized Project Funding on Solana

Innovative platform for secure, AI-driven project funding and management

Project Overview

FishTank is an innovative decentralized platform built on the Solana blockchain, designed to streamline and secure project funding. It uniquely combines AI-driven project analysis, NFT-based project representation, secure on-chain transactions, and comprehensive real-time tracking for both project creators and investors. The platform offers a robust ecosystem for managing project lifecycles, investments, and investor portfolios with a focus on security, transparency, and user experience.

Key Features

  • AI-Powered Project Analysis: Leverages OpenAI for in-depth project analysis and risk assessment, providing valuable insights to investors.
  • NFT-Based Project Representation: Projects are minted as NFTs with detailed metadata, stored decentrally on IPFS/NFT.Storage. On-chain ownership verification and management.
  • Secure & Transparent Funding: Utilizes Solana blockchain for secure and auditable funding transactions. "Project Tank" feature for managing and tracking funded projects.
  • Comprehensive Project & Investor Management: Multi-stage project lifecycle management. Real-time project status tracking and updates via WebSockets. Investor portfolio tracking with detailed metrics and transaction history.
  • Enhanced Security & Verification: KYC verification for project owners. Secure wallet integration (Phantom). Transaction signature verification.
  • Modern User Experience: Responsive UI built with React and Styled Components. Real-time updates and cross-device synchronization using IndexedDB caching. Comprehensive error handling and user feedback.

Technical Highlights & Architecture

  • Solana Blockchain Core: Custom Solana Program (FishTank Program ID: 2UAffCaPGycoxs5MJHduEhLS6B33N8rswbUwmrehHiwd on devnet) manages NFT minting, funding, and project state on-chain. Built with Anchor framework for smart contract development.
  • Decentralized Storage: IPFS and NFT.Storage for immutable and decentralized storage of project metadata and assets.
  • Real-time Capabilities: WebSocket integration (Socket.IO) for instant updates on project status, funding, and analytics.
  • Efficient Data Management: MongoDB backend for off-chain data, enabling complex queries, analytics, and user management.
  • AI Integration: OpenAI API integrated for automated project document analysis and risk assessment.

Tech Stack

  • Frontend: React.js 18.2, React Router DOM, Custom Hooks with Context API, Styled Components, React Icons, React Toastify, Dexie.js (IndexedDB), Socket.IO Client
  • Blockchain Integration (Solana): @solana/web3.js, @solana/spl-token, @metaplex-foundation/js (NFTs), @solana/wallet-adapter-react (Phantom, etc.), @project-serum/anchor (Smart Contracts)
  • Backend Services: Node.js, Express.js, MongoDB with Mongoose, Socket.IO, OpenAI API, PDF.js
  • Storage Solutions: NFT.Storage, IPFS HTTP Client
  • Development & Build: Craco, Babel, TypeScript, ESLint, Prettier, Docker (if used)
