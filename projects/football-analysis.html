AI-Powered Football Match Analysis with Advanced Player Tracking

Computer vision and machine learning for next-level sports analytics

Project Overview

This project leverages advanced computer vision and machine learning techniques to perform comprehensive football match analysis from video footage. It focuses on accurately detecting and tracking players, referees, and the ball using a custom-trained YOLO model. The system further assigns players to teams based on jersey color using K-means clustering and calculates key performance metrics such as team ball possession, player speed, and distance covered, by compensating for camera movement with Optical Flow and applying Perspective Transformation for real-world measurements.

Key Technical Achievements & Features

  • Robust Object Detection & Tracking: Utilized a custom-trained YOLO (v5) model for high-accuracy detection of players, referees, and the football. Implemented tracking algorithms to maintain object identities across frames.
  • Automated Team Assignment: Employed K-means clustering for pixel segmentation on player jerseys to automatically assign players to their respective teams based on dominant t-shirt colors.
  • Camera Motion Compensation: Applied Optical Flow techniques to measure and compensate for camera movement between frames, ensuring more accurate tracking of player movements relative to the pitch.
  • Real-World Metric Calculation: Implemented Perspective Transformation to map player positions from 2D video coordinates to a 3D real-world representation, allowing for movement measurement in meters rather than pixels.
  • Performance Analytics: Calculated key metrics including: Team ball possession percentage. Individual player speed. Total distance covered by each player.

Core Technologies Used

  • Primary Language: Python 3.x
  • Object Detection: Ultralytics (for YOLOv5)
  • Computer Vision: OpenCV
  • Clustering: Scikit-learn (for K-means, implied if not directly used via a library that wraps it)
  • Numerical Computation & Data Handling: NumPy, Pandas
  • Visualization: Matplotlib
  • Tracking & Utilities: Supervision (library)
